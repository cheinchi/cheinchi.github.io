Chapter 3 纳什均衡

纳什均衡Nash Equilibrum

def: 对于所有的$a_i\in A_i$存在$ u_i(a*i,a*{-i})\geq u_i(a_i,a*{-i})$

纳什均衡的特点

  • 纳什收益不等于纳什均衡本身
  • 纳什均衡是理性行为的结果
  • 纳什均衡是自动达到的过程
  • 学习的结果
  • 演化的结果

NE的奇数定理

几乎所有有限博弈都只有奇数个纳什均衡

  • 纯策略的NE
  • 混合策略的NE

弱占优策略

def: 对于所有的$a_i\in A_i$有任意的$ a’i\in A_i $ $u_i(a_i,a{-i})\geq u_i(a’i,a{-i})$同时存在 $ a_{-i}\in A_{-i}$ $ u_i(a_i,a_{-i})>u_i(a’i,a{-i})$

  • 对于弱占优策略来说，可能会丢失纳什均衡解

纳什均衡与ISD

Supermodular game超模博弈

简单描述：一个player的行为的边际价值是提升的在其他玩家的行为下；

对这样个G，我们有Topkis 定理：

a） G为supermodular game

b）S非空紧致

c) ui is upper semicontinuous in si for each s-i

完备格

Complete lattice 一种组合构形，满足：

它的任意一个非空子集均有最大元和最小元。

古诺模型

basic suppose:

  • homogenous product/good:
  • 市场容量：p=a-Q
  • Constant unit cost: c
  • 在NE的分析框架之下：

$$ N={1,2}; \A_1=A_2=[0,\infty]; \ \pi_i (q_1,q_2)=[a-(q_1+q_2)]q_i-cq_i $$

  • 对于BSF而言在cournot model内，就是将各自生产产品的利润最大化：

​ $$ \pi_i (q_1,q_2)=[a-(q_1+q_2)]q_i-cq_i $$

  • 目标函数：$ max_{q_i\in A_i} \pi_i(q_i,q_j*)=[a-(q_i+q_j*)]q_i-cq_i$

  • 一阶条件: $ a-2q_i-q_j*-c=0 \to q_i=1/2(a-a_j*-c)$

  • 古诺模型的解. $ q_1=q_2=\frac{a-c}{3}$

最优回应函数(Best Response Function,BSF)

对于给定的给定$a_{-i}$就可以得到BSF，构建了一个单射，但这从动态的过程去分析了静态博弈，即对于信息集而言，$ a_i$要对其进行回应，就是要通过分析对方的可能策略来进行应对。

$ B_i(a_{-i})={a_i\in A_i|u_i(a_i,a_{-i}\geq u_i(a’i,a{-i}) for {,}all{,}a’_i\in A_i }$

图解古诺模型

从两斜线不断通过相互逼近的方式来实现。
