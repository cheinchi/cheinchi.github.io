chapter 6 Extensive game with perfect info

Example 1: chain store game

  • a chain store(CS) has a branch in a city,

  • There is one potential competitor(C) in the city.

  • the game proceeds a follows:C decide whether to enter the market,Given C’s choice, CS decides whether to accommodate or fight back.

the profits are (0,0) (CS’s profit, C’s profit) if C enter and CS fights back, (2,2) is C enters and CS accommodates, and (5,1) if C does no enter.

this game can be des as upper. we could notice that in the description has imply a order to behave.

Formal Model

Extensive game with perfect info consists of

  • a finite set N
  • a set of sequence H such that
    • $\emptyset \in H$
    • $(a^1,\cdots,a^k)\in H\to (a^1,\cdots ,a^\ell)\in H ,\forall \ell<k$
  • a function P: $H/Z\to N$
  • a function $V_i: Z\to \mathfrak R \forall i \in N$

H is the set of history with

​ Z as the set of terminal histories and

​ $\empty$ as the initial history

P specifies who makes a choice at each history.

$V_i(Z)$ is player i’s payoff at terminal history z.

at history $h\in H/Z$,the set of actions that are available to player $P(h)$ is $$ A(h)={a|(h,a)\in H} $$

  • A stratrgy is not just a contingent(队列) plan of actions. It specifies an action at every history, even at histories that are never reached given the strategy.

History

  • 某一特定策略组合的发生（包含在整个的game之中的）

  • a period t history $h_t=(a_1,\cdots,a_{t-1})\in H_t=A^{t-1}$ is a sequence of the past action profiles at the beginning of period t.

  • the initial history is $H_1={0}$by convention

  • $H=\bigcup^\infty_{t=1}H_t$ is the set of all such histories

  • subhistory：at a history subset，nodes that be included is no more than whole history

  • in game theory ,a better way to describe uncertainty is using dynamic adding the dimention of time.

subgame

Def：属于一个博弈的一部分，可以被单独拿来分析；策略、信息独立；

一个扩展式博弈的subgame G的单个节点x和所有他的成功者T，$ x’\in G $ and $ x^"\in h(x’)$对于$ x^{’’}\in G$

  • 信息集，行为和payoff在子博弈中是被继承；

  • 在动态博弈中，所有参与人先后都采取了一次行动后所构成的一组新的博弈 ，这组博弈中的每一个都称为“子博弈”。

进一步分析subgame perfect equilibrium:

Backward Induction

Find all nonterminal histories of L-1 length and assign an optimal action there. Eliminate unreached L-length terminal histories and regard other L-length terminal histories as L-1-length terminal histories .

Subgame Perfect Equilibrium

At any history, the ”remaining game “(在这个history之后所形成的subgame）can be regard as an extensive game on its own. It is called a subgame

the SPE satisfies the following conditions

  • $h’ \in H|_h \Leftrightarrow (h,h’)\in H$
  • $P|_h(h’)=p(h,h’)\forall h’\in H|_h$
  • $V_{i}|_h(h’)=V_i(h,h’)\forall h’\in Z|_h\subset H|_h$

note that a strategy specifies a strategy for any subgame.

SPE(subgame perfect equilibrium)子博弈精炼纳什均衡表示在所有的子博弈（包括原博弈）中占优的策略

  • $ \sigma$:是博弈T的策略；

  • $ \sigma|G$: 属于策略组合induced by $ \sigma$在T的子博弈G中；

核心思想：将纳什均衡中不可信威胁剔除出NE中，可以得到更为精确地描述NE的方式。

  • 要求参与者的决策在任何时点上都是最优的，决策者要“随机应变“。

  • 序贯理性：指不论过去发生了什么，参与人应该在博弈的每个时点上最优化自己的策略。子博弈精炼纳什均衡所要求的正是参与人应该是序惯理性的。

  • 有限的完美信息中存在SPE

One-Shot Deviation Principle

a incentive constraints for finite horizon extensive games with perfect information.

one-shot deviation from $s_i$ if $s’_i$ prescribes a different action only at the initial history for any $h’\neq \empty$

Continuity at Infinty

In general, one-shot deviation doesn’t hold form infinite horizon games.

continuity at infinity if payoffs in very far future are not important(use discount )

Every finite horizon extensive game satisfies continuity at infinity. $$ V_i(z)=\sum^\infty_{t=0}\delta^tg_i(a^t) $$
